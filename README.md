## 🚀 Como rodar o projeto localmente

Clone o repositório e rode o projeto em sua máquina 👇

```bash
# Clone o repositório
git clone https://github.com/DiegoBY/myportfolio.git

# Entre na pasta
cd portfolio

# Instale as dependências
npm install

# Rode o servidor de desenvolvimento
npm run dev
```

Após isso, o projeto estará disponível em:
👉 http://localhost:5173
